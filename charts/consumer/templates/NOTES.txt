Consumer deployed in namespace {{ .Release.Namespace }}.

To check the consumer logs, run:
  kubectl logs -n {{ .Release.Namespace }} -l app={{ include "consumer.name" . }}

To connect to RabbitMQ, make sure the RabbitMQ service is accessible at:
  Host: {{ .Values.env.RABBITMQ_HOST }}
  Port: {{ .Values.env.RABBITMQ_PORT }}

If you want to port-forward for debugging:
  POD_NAME=$(kubectl get pods -n {{ .Release.Namespace }} -l app={{ include "consumer.name" . }} -o jsonpath="{.items[0].metadata.name}")
  kubectl port-forward -n {{ .Release.Namespace }} $POD_NAME 5672:{{ .Values.env.RABBITMQ_PORT }}
